## 🎯 Description
EDEN Frontend est la maquette interactive du chatbot d'orientation EDEN. Le projet simule un test de 5 questions avec algorithme JavaScript pour recommander une filière aux élèves congolais. Tout tourne côté client.
 

## ✨ Fonctionnalités
- Chat interactif avec questions à choix multiples via `<select>`
- 5 étapes de questionnaire d'orientation
- Algorithme JS de recommandation de filière basé sur les réponses
- Design responsive mobile/desktop
- Animations CSS + scroll automatique du chat
- Zéro dépendance : Vanilla JS uniquement

## 🛠️ Technologies
- **HTML5** : Structure
- **CSS3** : Design + dégradés + glassmorphism 
- **JavaScript ES6** : Logique du chatbot + DOM
